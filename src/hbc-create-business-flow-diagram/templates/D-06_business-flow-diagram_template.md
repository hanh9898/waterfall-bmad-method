# D-06 Business Flow Diagram

## Business Flow of (Process Name)

### AS-IS (Current Flow)
```mermaid
sequenceDiagram
    actor User as User
    participant System as System

    User->>System: Action
    System-->>User: Result
```

### TO-BE (Target Flow)
```mermaid
sequenceDiagram
    actor User as User
    participant NewSystem as New System

    User->>NewSystem: New Action
    NewSystem-->>User: Improved Result
```
